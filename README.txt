Voordat het spel gespeeld en opgestard kan worden moeten de node modules aanwezig zijn. 
Dit kan gedaan worden door in een cmd venster de opdracht "npm install" te geven.

Pak het, door ons ingeleverde, bestand uit en open de map.
Voer de volgende toetsen reeks uit in deze map. LShift + rechtermuisknop. 
Klik op "Opdrachtvenster hier openen"
In dit venster moet het commande "node.exe server.js" gegeven worden. Als dit gedaan is, is de server gestart.

Open het bestand splash.html.

Veel plezier met het spelen van ons teken spel!
